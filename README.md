# X Platinum (XP)  

Enterprise real estate tokenization protocol for regulated, asset-backed digital securities.  

## Overview  
XP is an Ethereum-based token representing fractionalized ownership and control of real-world real estate assets, designed for institutional compliance and long-term governance.  

## Key Features  
- ERC-20 compatible with ERC-1404 compliance  
- Whitelisted transfers only  
- Issuer-controlled minting and burning  
- Investor read-only transparency  
- Upgradeable architecture
